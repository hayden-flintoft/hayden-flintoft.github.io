The CSS Box Model

What is the difference between Margin, Border, and Padding?

2024-06-19

When styling HTML content, we wrap the element in a box. This box contains a border which generally would be considered the outside edge of the element. Inside the border, we have padding. Padding is the space between the border and the content of the element. Padding is used to create space between the content and the border. Margin is the space outside the border. Margin is used to create space between the border and other elements on the page.

It is important to consider the box model when styling an element. The box model is the combination of the content, padding, border, and margin of an element. The box model is used to determine the size and position of an element on the page.

When measuring an object, we have two choices:

  • Outer dimensions: The outer dimensions of an object include the padding, border, and margin of the object.
  • Inner dimensions: The inner dimensions of an object include only the content of the object.

By default the object size only includes the content but we can change this by setting the box-sizing property to border-box to include the padding and boarder in the size.
